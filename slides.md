# Let's start with some examples
### Variables
```python
>>> a = "ceci est un"
>>> b = "test"
>>> print(a, b)
ceci est un test
```

### Conditions
```python
>>> if 5 < 10:
>>>     print("Yep")
>>> else:
>>>     print("Nope")
Yep
```

### Loops
```python
>>> for i in ["one", "two", "three", "four"]:
>>>     print(i)
one
two
three
four
```

### Calculus
```python
>>> 2 + 2
4
>>> 50 - 5*6
20
>>> (50 - 5*6) / 4
5.0
>>> 8 / 5
1.6
```
